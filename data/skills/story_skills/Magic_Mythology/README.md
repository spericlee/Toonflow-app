# 魔幻神话 · 导演叙事手法技能包

## 简介

本技能包为 **魔幻神话** 类型故事提供一套完整的导演叙事手法参考，涵盖从宏观叙事规划到微观分镜执行的全流程指导。适用于任何视觉风格。

## 核心理念

- **神话即人性** — 神话是人类集体潜意识的映射，众神的争斗、英雄的试炼、凡人的抉择，归根结底都是人性的放大与抽象。神性越强，人性越真
- **宿命与自由意志的角力** — 魔幻神话的核心驱动力是"预言能否被改变"：英雄明知命运却依然前行、凡人反抗神明、预言自我实现。宿命的沉重与意志的光芒并存
- **史诗感来自牺牲** — 魔幻神话中的高光时刻永远与牺牲绑定：牺牲安逸、牺牲所爱、牺牲生命。牺牲的规模决定了史诗的厚度
- **神话的尺度，人的温度** — 世界观可涵盖天地宇宙、诸神众魔，但叙事落脚点始终是"一个人的选择"。宏大的神话背景最终是为了照亮一个凡人的心灵

## 技能文件说明

### 1. 叙事规划手法 (`director_planning_narrative.md`)

导演在 **叙事规划阶段** 使用的技法参考，定义魔幻神话在主题立意、史诗节奏、场景神话氛围与声音方向上的整体规划方法。

| 章节 | 内容概要 |
|---|---|
| 主题立意与神话内核 | 神话类型本质、宿命与自由意志、光明与黑暗的辩证、诸神与凡人的关系 |
| 叙事结构与史诗节奏 | 英雄之旅、牺牲与高潮、预言叙事、多线并行策略 |
| 分场景氛围设计 | 六大典型神话段落设计、神话空间叙事、圣光与暗影隐喻 |
| 声音与音乐方向 | 交响史诗、合唱赞美诗、民族古乐、寂静神圣 |
| 构图与景别叙事 | 天地尺度全景/神性特写/巨物对比/倾斜失序 |
| 镜头运动与节奏 | 史诗缓推、神性上摇、战斗冲刺、圣光拉远 |

### 2. 分镜表叙事手法 (`director_storyboard_table_narrative.md`)

导演在 **分镜表制作阶段** 使用的技法参考，定义魔幻神话在分镜表中的景别策略、运镜节奏、时长把控、神话行动分镜与转场手法。

| 章节 | 内容概要 |
|---|---|
| 分镜表定位 | 分镜表作为神话叙事核心工具的定位说明 |
| 景别选择 | 天地远景价值、神性特写、巨物/圣物景别 |
| 运镜节奏 | 史诗缓推感、神性上摇、危机冲刺、神圣缓移 |
| 时长把控 | 神迹凝视时长、战斗压缩、沉默神圣的留白 |
| 神话行动设计 | 试炼/战斗/祭祀的分镜逻辑、神器争夺镜头语言 |
| 神性抉择叙事 | 道德困境镜头、信仰与背叛、牺牲的仪式感 |
| 台词与留白 | 箴言如诗、环境叙事替代对白、预言碎片 |
| 转场设计 | 时空蒙太奇/圣光/梦境传送/闪回神话 |

## 使用方式

本技能包作为 ToonFlow 导演 AI 的叙事手法参考，在故事创作流程中自动加载，指导：

1. **叙事规划** — 确定神话内核、史诗节奏曲线、场景神话氛围与音效方向
2. **分镜表生成** — 根据叙事规划输出具体的景别、运镜、时长与转场方案

两个文件分工明确、逐层递进，从"为什么这样拍"到"具体怎么拍"形成完整闭环。
