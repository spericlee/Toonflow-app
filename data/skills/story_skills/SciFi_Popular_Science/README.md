# 科普科幻 · 导演叙事手法技能包

## 简介

本技能包为 **科普科幻** 类型故事提供一套完整的导演叙事手法参考，涵盖从宏观叙事规划到微观分镜执行的全流程指导。适用于任何视觉风格。

## 核心理念

- **科学即叙事** — 科学本身充满戏剧性：一个假说的诞生、一次实验的突破、一个未知现象的发现，其张力不亚于任何虚构冲突。科学探索的过程就是最好的故事
- **理性与情感的辩证** — 科普科幻的核心张力是理性与情感的角力：科学家面对数据与直觉的冲突、知识与道德的抉择、客观真理与个人信念的碰撞
- **未知是最大的悬念** — 科普科幻中最大的悬念不是"谁赢了"，而是"答案是什么"。对未知的好奇、对真理的探索，是驱动叙事的根本力量
- **科学的温度** — 科学不是冰冷的公式，而是人类对世界最炽热的好奇心。越理性的探索，越需要人性的温度来照亮

## 技能文件说明

### 1. 叙事规划手法 (`director_planning_narrative.md`)

导演在 **叙事规划阶段** 使用的技法参考，定义科普科幻在主题立意、探索节奏、场景科学氛围与声音方向上的整体规划方法。

| 章节 | 内容概要 |
|---|---|
| 主题立意与科学内核 | 科学探索精神、理性与情感的平衡、未知与发现的辩证 |
| 叙事结构与探索节奏 | 假说驱动、实验验证、阶段性突破、挫折与重新出发 |
| 分场景氛围设计 | 六大典型科普段落设计、科学空间氛围、自然奇观与科技奇观 |
| 声音与音乐方向 | 科学仪器声、自然之声、钢琴与弦乐、沉默的思考 |
| 构图与景别叙事 | 探索全景/专注特写/人与科技对比/发现构图 |
| 镜头运动与节奏 | 稳定纪实探索、发现缓推、实验跟拍、思考静止 |

### 2. 分镜表叙事手法 (`director_storyboard_table_narrative.md`)

导演在 **分镜表制作阶段** 使用的技法参考，定义科普科幻在分镜表中的景别策略、运镜节奏、时长把控、科学探索行动分镜与转场手法。

| 章节 | 内容概要 |
|---|---|
| 分镜表定位 | 分镜表作为科学叙事核心工具的定位说明 |
| 景别选择 | 探索远景价值、专注特写、科技设备景别 |
| 运镜节奏 | 稳定纪实感、发现缓推、实验跟拍、思考静止 |
| 时长把控 | 科学观测时长、发现停顿、思考的留白 |
| 探索行动设计 | 实验/观察/发现的分镜逻辑、科学发现镜头语言 |
| 科学抉择叙事 | 道德困境镜头、信仰与真相、突破的仪式感 |
| 台词与留白 | 精准表达、数据叙事替代夸张对白、发现时刻的沉默 |
| 转场设计 | 时空蒙太奇/数据流动/镜头类比/发现闪回 |

## 使用方式

本技能包作为 ToonFlow 导演 AI 的叙事手法参考，在故事创作流程中自动加载，指导：

1. **叙事规划** — 确定科学内核、探索节奏曲线、场景科学氛围与音效方向
2. **分镜表生成** — 根据叙事规划输出具体的景别、运镜、时长与转场方案

两个文件分工明确、逐层递进，从"为什么这样拍"到"具体怎么拍"形成完整闭环。
