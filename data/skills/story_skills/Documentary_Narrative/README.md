# 纪录讲述 · 导演叙事手法技能包

## 简介

本技能包为 **纪录讲述** 类型故事提供一套完整的导演叙事手法参考，涵盖从宏观叙事规划到微观分镜执行的全流程指导。适用于任何视觉风格。

## 核心理念

- **真实即叙事** — 真实本身拥有最强大的叙事力量。一个真实人物的真实经历，比任何虚构情节都更有感染力
- **观察者视角** — 纪录片导演不是故事的创造者，而是真实的观察者和提炼者。镜头语言应当服务于"呈现真实"而非"制造戏剧"
- **人文关怀** — 纪录讲述的核心是对人的关注：人的命运、人的情感、人与自然和社会的互动
- **真相的尊严** — 尊重事实、尊重被摄对象、尊重观众，是纪录讲述的伦理基石

## 技能文件说明

### 1. 叙事规划手法 (`director_planning_narrative.md`)

导演在 **叙事规划阶段** 使用的技法参考，定义纪录讲述在主题立意、叙事节奏、场景纪实氛围与声音方向上的整体规划方法。

### 2. 分镜表叙事手法 (`director_storyboard_table_narrative.md`)

导演在 **分镜表制作阶段** 使用的技法参考，定义纪录讲述在分镜表中的景别策略、运镜节奏、时长把控、纪实行动设计与转场手法。
