# 真人科学纪实风格说明

本风格专为"真人科学纪实"题材打造，所有美术提示词、规范和生成内容均严格限定于：

- **科学纪实世界观**：以真实世界为背景，涵盖自然科学、人文地理、历史考古、社会生活等纪实题材，杜绝魔幻/科幻/架空元素。
- **真人写实**：所有角色、场景、道具均以真人写实摄影为标准，强调五官、皮肤、发丝、材质等极致细节与真实质感。
- **纪实美学**：在写实基础上追求纪录片级真实质感，画面自然真实，细节丰富，整体氛围真实中见温度、客观中蕴人文。
- **真实之美**：适合展现自然奇观、人文故事、科学探索、历史踪迹等纪实内容，强调客观视角下的真实情感表达。

## 适用范围

- 真人科学纪实题材的AI美术生成（角色、场景、道具、分镜等）
- 需要极致写实、细腻质感、纪录片级纪实氛围的视觉创作

## 严禁内容

- 卡通、动漫、插画、二次元风格
- 魔幻/奇幻/科幻/古风架空元素
- 高饱和荧光色、霓虹色、赛博朋克色彩
- 暗示性、暴露、透视等不适宜内容

## 风格体验

在本风格下，您将体验到：

- 角色面容真实自然、气质朴实或专注，服饰为现代日常/专业纪实风格
- 场景空间层次丰富，材质纹理超清晰，氛围真实而富有故事感
- 道具、工具、设备等均为真实世界造型，质感极致
- 分镜与视频均以真人写实为基础，强调纪实叙事与真实之美的表达

---

本风格仅适用于"真人科学纪实"相关项目。如需其他风格（如插画、动漫、奇幻等），请勿使用本套提示词。
