# 人物衍生资产生成 · 约束手册

---

## 一、叠加原则

1. **面容不变** — 叠加后五官必须与底模完全一致
2. **姿态不变** — 保持底模自然站立姿态
3. **逐层可控** — 每层独立描述
4. **风格统一** — 所有服化元素服从纪实美学
5. **纯服化范畴** — 仅叠加妆容/发型/服饰/配饰

---

## 二、叠加层级

| 层级 | 内容 | 说明 |
|---|---|---|
| L0 | 底模 | 基础形象底模 |

| L2 | 发型造型 | 扎发/盘发/披散 |
| L3 | 内搭 | 替换基础内衬 |
| L4 | 外衣/主服 | 户外夹克/工作服/日常外套 |
| L5 | 配饰 | 眼镜/帽子/围巾/背包 |

## 九、约束规则

### 必守

| 编号 | 规则 |
|---|---|
| R1 | 叠加后面容必须与底模一致 |
| R2 | 服饰必须真实自然 |
| R3 | 必须输出四视图 |
| R4 | 必须指定「纯净中性灰背景」 |

### 严禁

| 编号 | 严禁 |
|---|---|
| X1 | 妆容过于浓艳夸张 |
| X2 | 服饰不真实（奇幻/科幻） |
| X3 | 输出提示词以外内容 |
| X4 | 加入道具交互 |
| X5 | 改变底模姿态 |
