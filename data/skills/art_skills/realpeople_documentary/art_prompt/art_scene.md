# 场景图生成 · 约束手册

---

## 一、场景美学原则

1. **真实叙事** — 场景承载真实世界的叙事功能
2. **层次纵深** — 所有场景必须具备前/中/后景
3. **质感至上** — 自然/人工材质纹理必须超清晰
4. **实拍为锚** — 以真实摄影为标准，追求镜头光学特征与物理光照

---

## 二、季节色调映射

| 季节 | 主色调 | 辅色调 |
|---|---|---|
| 春 | 嫩绿 + 花彩色 | 天光白、远山蓝 |
| 夏 | 浓绿 + 天蓝 | 草木绿、素白 |
| 秋 | 金黄 + 暖橙 | 大地褐、暖阳黄 |
| 冬 | 霜白 + 灰蓝 | 深炭黑、金属灰 |

## 三、室内场景

### 空间规范

| 维度 | 约束 | 提示词 |
|---|---|---|
| 风格 | 真实日常空间 | 真实{空间类型}风格 |
| 材质 | 木质/布料/金属/玻璃/石材 | 木质家具、布料质感 |
| 光照 | 自然光为主，真实光照 | 自然光漫射、窗光 |
| 镜头感 | 浅景深虚化 | shallow depth of field |

### 室内类型速查

| 类型 | 核心元素 | 氛围词 |
|---|---|---|
| 民居/家 | 家具、窗、灯具、家庭物品 | 温馨真实、生活气息 |
| 研究室/书房 | 书架、书桌、电脑、笔记 | 安静专注、知识氛围 |
| 博物馆/展览馆 | 展柜、展品、照明 | 文化沉淀、历史厚重 |
| 工坊/车间 | 工具台、器械、材料 | 真实劳作、手工艺美 |
| 自然博物馆 | 标本柜、模型、解说牌 | 科学氛围、自然博物 |

## 四、室外场景

### 空间规范

| 维度 | 约束 | 提示词 |
|---|---|---|
| 类型 | 自然景观/乡村/城市/野外 | {场景}，{季节}，{时间} |
| 天候 | 真实天候 | 晴/阴/雾/雨/雪 |
| 植被 | 真实地域植被 | 自然植被 |
| 水体 | 真实水体光影 | 水面波光 |
| 光线 | 自然光唯一光源 | 自然光照、体积光 |
| 瑕疵感 | 自然风化痕迹 | 自然痕迹、真实质感 |

### 室外类型速查

| 类型 | 核心元素 | 氛围词 |
|---|---|---|
| 山林/森林 | 树木、溪流、苔藓、光影 | 自然幽静、绿意盎然 |
| 田野/乡村 | 农田、农舍、远山 | 田园开阔、宁静真实 |
| 海岸/湖泊 | 水面、沙滩、天空 | 辽阔宁静、水天一色 |
| 城市街巷 | 建筑、行人、店铺 | 市井真实、生活烟火 |
| 古建/遗址 | 石墙、残垣、雕刻 | 岁月沉淀、历史痕迹 |

## 五、主视图规范

| 项目 | 约束 |
|---|---|
| 布局 | 单画面 |
| 人物 | **严禁出现任何人物** |
| 光线 | 单一光源逻辑 |
| 画面比例 | 默认 16:9 |

```

```

## 七、约束规则

### 必守

| 编号 | 规则 |
|---|---|
| R1 | 场景必须有「前中后景层次」 |
| R2 | 室外必须包含「空气透视」 |
| R3 | 场景图必须为「单画面」 |
| R4 | **严禁出现任何人物** |
| R5 | 必须包含实拍摄影关键词 |

### 严禁

| 编号 | 严禁 |
|---|---|
| X1 | 背景纯白/纯黑/无场景 |
| X2 | 场景无纵深 |
| X3 | 出现任何人物 |
| X4 | 3D 渲染/CG 动画质感 |
