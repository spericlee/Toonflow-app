# 道具图像生成 · 约束手册

---

## 一、道具设计原则

1. **功能可读** — 道具用途一目了然，造型服务于功能
2. **质感极致** — 材质纹理必须清晰可辨（金属/玻璃/塑料/陶瓷/木材）
3. **科技一致** — 所有道具必须符合现代/科技世界观，禁止魔幻/古风元素
4. **尺度明确** — 通过参照物或标注暗示道具真实尺寸
5. **纯道具独立展示** — 画面中只能出现道具本身，严禁出现任何人物、手部、肢体，道具不可处于被持有/佩戴/握持状态，必须以静物陈列方式独立呈现

## 三、多角度设定图规范

### 视图定义

| 位置 | 视图 | 角度 | 要求 | 提示词 |
|---|---|---|---|---|
| 左上 | 正面图 | 正面 0° | 道具完整正面形态 | front view |
| 右上 | 侧面图 | 侧面 90° | 厚度/轮廓/结构清晰 | side view |
| 左下 | 背面图 | 背面 180° | 道具背部结构/接口 | back view |
| 右下 | 细节特写 | 局部放大 | 材质纹理/工艺细节 | detail closeup |

### 画面规范

| 项目 | 约束 |
|---|---|
| 布局 | 同一画面四宫格（2×2），上下左右四视角 |
| 背景 | 纯净中性灰 #E8E8E8 |
| 光线 | 均匀柔光，无硬阴影 |
| 比例 | 每格道具占格内主体 70%+ |
| 投影 | 允许自然地面微投影 |
| 画面比例 | 建议 1:1 |

## 四、材质渲染约束

| 材质 | 渲染要求 | 提示词 |
|---|---|---|
| 金属 | 反光/高光/冷光泽、拉丝或镜面 | 金属质感、冷光泽、反光清晰 |
| 玻璃 | 通透、折射、反光 | 玻璃通透、折射清晰 |
| 塑料 | 磨砂或亮面、质感均匀 | 塑料质感、表面均匀 |
| 陶瓷 | 釉面光泽、色泽均匀 | 陶瓷质感、釉面光泽 |
| 木材 | 木纹清晰、年轮可见 | 木纹清晰、质感温润 |
| 织物/布料 | 纤维质感、编织纹理 | 布料纹理、编织细腻 |

## 六、约束规则

### 必守

| 编号 | 规则 |
|---|---|
| R1 | 必须指定「纯净中性灰背景」 |
| R2 | 必须明确道具材质与工艺 |
| R3 | 道具造型必须符合现代/科技世界观 |

### 严禁

| 编号 | 严禁 |
|---|---|
| X1 | 复杂场景背景 |
| X2 | 道具与人物同画面（本环节为纯道具图） |
| X3 | 出现任何人物形象，包括全身、半身、局部（手、手指、手臂等肢体） |
| X4 | 道具处于被持有、握持、佩戴、使用中的状态 |
| X5 | 出现暗示人物存在的元素（如手持痕迹、佩戴视角、使用姿态） |
