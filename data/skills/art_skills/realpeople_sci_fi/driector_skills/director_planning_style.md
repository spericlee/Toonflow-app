# 科谱约束 · 真人科谱写实 · 技法参考

---

## 一、色调体系与画面基调

- **色调基底** — 全片以科技蓝（C1）、自然肤（C2）、冰湖蓝（C6）为基底色，整体色温中性偏冷（5000-5500K），饱和度中低（30-50%），呈现理性明亮的高级自然调
- **冷暖叙事对比** — 暖色（暖光黄 C7、生机绿 C5）作为叙事转折的视觉信号，用于情感回暖段落局部点缀。冷暖切换应与故事弧线同步，而非随意混用
- **冷暖分配策略** — 理性/客观/冷静段落严格维持中性偏冷基底，禁用暖橙色调；暖色仅在情感回暖后释放，用冷暖对比做叙事转折
- **色盘先行原则** — 段落规划需先绑定情绪场景（初次相遇/升温/分离/重逢/科学发现等），再确定主色+辅色与光影方案，避免"剧情对了但情绪不对色"
- **禁用色域** — 高饱和荧光色、霓虹色、赛博朋克色彩、魔幻色彩体系均与本风格不兼容

## 二、光影方案体系

- **光影即叙事** — 7 套光影方案对应不同情绪段落，导演规划阶段应在段落层面确定光影基调方向，而非逐镜指定
- **光源角度** — 正给镜头默认 38.5° 斜射（非 45°），侧给镜头可适当调整

| 光影方案 | 方案名 | 色调倾向 | 适用情绪 |
|---|---|---|---|
| A | 自然柔光 | 中性白底 + 微暖肤光 | 日常温馨、轻松甜蜜 |
| B | 窗光侧映 | 科技蓝+ 边缘光 | 专注工作、知性时刻 |
| C | 台灯暖影 | 暖光黄主导 + 深空黑暗部 | 夜间暧昧、亲密升温 |
| D | 月光冷辉 | 冰湖蓝 + 月华白 | 夜间孤寂、思念独处 |
| E | 天光漫射 | 中性白 + 侧光斑驳 | 室内日间、日常起居 |
| F | 屏幕光映 | 科技蓝光源 + 银白反光 | 科技场景、屏幕前专注 |
| G | 冷光侧射 | 冰湖蓝冷白 + 38.5°侧射冷光，无暖色 | 分离远行、理性克制冷清 |

- **冷暖光分配** — 暖色光（暖光黄）适用于情感回暖段落；理性/分离段落一律中性偏冷。导演可根据叙事需要调整冷暖切换点
- **氛围方向映射** — 每场戏的氛围方向应能映射到上述光影方案（A-G）的某一方向，确保视觉一致性

## 三、质感方向

- **超清纪实感** — 真人写实摄影的核心：毛孔可见、发丝根根分明、纹理细节超清晰
- **强对比度 + 极致细节** — 这是画面质感的锚点。不是胶片颗粒，不是艺术滤镜，是纪录片级科学摄影
- **材质真实** — 所有服化道的材质必须可信：布料纹理、金属光泽、玻璃通透、纸张质感。禁止"塑料感"和"CG感"
- **皮肤质感** — 真实皮肤的毛孔、微瑕、光泽感。既不过度磨皮也不刻意放大瑕疵，追求"真实但好看"
- **写实不等于平淡** — 真人科谱写实强调真实可拍的影像表达（真实光位、真实材质、真实动作节奏），通过镜头与情绪设计放大感染力，而非依赖奇观特效

## 四、科谱场景空间元素

科谱科幻世界观特有的场景元素及其视觉叙事功能：

- **窗户/玻璃幕墙/天窗** — 天然的框架式构图道具，引入自然光与外界景色，制造"通透感"与室内外联系
- **实验台/书架/仪器柜** — 知识氛围的载体，丰富的细节填充画面，传递"科学/专注/专业"的空间气质
- **屏幕/投影/数据可视化** — 科技感的视觉元素，屏幕光可作为场景光源，数据流动增加"科技叙事"感
- **段落间用场景空镜过渡** — 本风格有丰富的场景资产（不同时段/天候变体），段落衔接建议用场景空镜做情绪缓冲，不要硬切
- **转折点用视觉而非台词** — 优先使用画面手段（光影变化、景别跳切、空镜隐喻）而非依赖对白解释

## 五、科谱声音方向

科谱科幻世界观下的声音元素约束：

### 乐器选择

- **钢琴** — 理性/温暖段落的基底乐器，清澈的琴音最能表现思考与专注
- **弦乐（小提琴/大提琴）** — 情感段落的主旋律乐器，温暖而不过度煽情
- **木管（单簧管/长笛）** — 自然/科学探索段落，轻盈通透
- **电子氛围** — 科技场景/太空段落的基底，低音铺底+微弱的电子音色
- 弦乐铺底可增加电影感但不宜喧宾夺主

### 乐器组合策略

| 情绪阶段 | 乐器组合 |
|---|---|
| 平稳/思考/阅读 | 钢琴独奏 或 钢琴 + 弦乐铺底 |
| 探索/发现 | 长笛 + 钢琴 + 电子氛围 |
| 情感高潮/重逢 | 弦乐 + 钢琴 |
| 科技/太空 | 电子氛围 + 低音弦乐 |

### 科谱环境音

- **典型环境音层次** — 实验室仪器嗡鸣 / 键盘敲击声 / 纸张翻动 / 脚步声 / 自然风声 / 鸟鸣虫唱 / 液体流动声
- **每场戏标注 1-2 个核心环境音**，帮助后续音效设计。环境音层次越丰富，科谱场景越有沉浸感
