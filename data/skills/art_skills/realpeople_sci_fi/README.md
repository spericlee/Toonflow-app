# 真人科谱科幻风格说明

本风格专为"真人科谱科幻"题材打造，所有美术提示词、规范和生成内容均严格限定于：

- **科谱科幻世界观**：以现代科学为基础、未来科技为延伸的写实科幻世界观，涵盖太空探索、实验室、自然奇观、科技设施等，杜绝魔幻/古风/奇幻元素。
- **真人写实**：所有角色、场景、道具均以真人写实摄影为标准，强调五官、皮肤、发丝、材质等极致细节与真实质感。
- **科谱写实纪实**：在写实基础上追求纪录片级科学质感，画面清晰明亮，细节丰富，整体氛围理性中见温度、科技中蕴人文。
- **科学浪漫**：适合展现人类探索未知、科技改变命运、自然与科学之美的故事，强调理性光辉下的情感表达。

## 适用范围

- 真人科谱科幻题材的AI美术生成（角色、场景、道具、分镜等）
- 需要极致写实、细腻质感、科学纪实氛围的视觉创作

## 严禁内容

- 卡通、动漫、插画、二次元风格
- 魔幻/奇幻/古风/神话元素
- 高饱和荧光色、霓虹色、赛博朋克式霓虹
- 暗示性、暴露、透视等不适宜内容

## 风格体验

在本风格下，您将体验到：

- 角色面容精致、气质理性或专注，服饰为现代/未来科技写实风格
- 场景空间层次丰富，材质纹理超清晰，氛围真实且富有科学美感
- 道具、仪器、设备等均为科技造型，工艺精细，质感极致
- 分镜与视频均以真人写实为基础，强调叙事与科学之美的表达

---

本风格仅适用于"真人科谱科幻"相关项目。如需其他风格（如插画、动漫、奇幻等），请勿使用本套提示词。
