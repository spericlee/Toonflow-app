# 魔幻约束 · 真人魔幻写实 · 技法参考

---

## 一、色调体系与画面基调

- **色调基底** — 全片以暮光紫（C1）、冷瓷肤（C2）、星辰蓝（C6）为基底色，整体色温偏冷（4800-5200K），饱和度中低（35-55%），呈现神秘华美的高级灰调
- **冷暖叙事对比** — 暖色（琥珀暖 C7、秘银光 C3、精灵绿 C5）作为叙事转折的视觉信号，用于情感回暖段落局部点缀。冷暖切换应与故事弧线同步，而非随意混用
- **冷暖分配策略** — 悲痛/孤绝/压抑段落严格维持冷色调基底，禁用暖橙、琥珀等暖色调；暖色仅在情感回暖后释放，用冷暖对比做叙事转折
- **色盘先行原则** — 段落规划需先绑定情绪场景（宿命相遇/升温/误会/魔法觉醒/重逢等），再确定主色+辅色与光影方案，避免"剧情对了但情绪不对色"
- **禁用色域** — 高饱和荧光色、霓虹色、现代色彩体系均与本风格不兼容

---

## 二、光影方案体系

- **光影即叙事** — 7 套光影方案对应不同情绪段落，导演规划阶段应在段落层面确定光影基调方向，而非逐镜指定
- **光源角度** — 正给镜头默认 38.5° 斜射（非 45°），侧给镜头可适当调整

| 光影方案 | 方案名 | 色调倾向 | 适用情绪 |
|---|---|---|---|
| A | 林隙柔光 | 冷瓷底 + 微暖肤光 | 日常温馨、轻松甜蜜 |
| B | 侧逆圣光 | 秘银光 + 边缘光 | 神圣亮相、惊艳登场 |
| C | 壁炉暖影 | 琥珀暖主导 + 暗夜黑暗部 | 夜间暧昧、亲密升温 |
| D | 月光冷辉 | 星辰蓝 + 月华银 | 夜间孤寂、思念独处 |
| E | 窗纱透光 | 冷瓷底 + 侧光斑驳 | 室内日间、日常起居 |
| F | 魔法漫射 | 暮光紫远景 + 秘银光雾气 | 魔法场景、意境空镜 |
| G | 冷光侧射 | 星辰蓝冷白 + 38.5°侧射冷光，无暖色 | 悲痛孤绝、压抑冷清 |

- **冷暖光分配** — 暖色光（琥珀暖/暖橙）适用于情感回暖段落；悲痛/孤绝段落一律冷光。导演可根据叙事需要调整冷暖切换点
- **氛围方向映射** — 每场戏的氛围方向应能映射到上述光影方案（A-G）的某一方向，确保视觉一致性

## 三、质感方向

- **超清纪实感** — 真人写实摄影的核心：毛孔可见、发丝根根分明、纹理细节超清晰
- **强对比度 + 极致细节** — 这是画面质感的锚点。不是胶片颗粒，不是柔焦写意，是史诗级纪实摄影
- **材质真实** — 所有服化道的材质必须可信：丝绒的光泽、皮革的纹理、金属的反光、水晶的通透。禁止"塑料感"和"CG感"
- **皮肤质感** — 真实皮肤的毛孔、微瑕、光泽感。既不过度磨皮也不刻意放大瑕疵，追求"真实但好看"
- **写实不等于平淡** — 真人魔幻写实强调真实可拍的影像表达（真实光位、真实材质、真实动作节奏），通过镜头与情绪设计放大感染力，而非依赖奇观特效

## 四、魔幻场景空间元素

魔幻世界观特有的场景元素及其视觉叙事功能：

- **拱门/廊柱/彩色玻璃窗** — 天然的框架式构图道具，制造"神圣感"的层次感与空间纵深
- **古树/藤蔓/魔法光点** — 留白构图的天然载体，景即情：古木参天 = 神秘、魔法光点 = 觉醒、落叶纷飞 = 离愁
- **壁炉/烛台/魔法灯** — 魔幻世界的光源载体，炉火 = 暖/私密（方案C）、魔法灯 = 冷/神秘（方案D/F）、烛台 = 日常/静谧（方案E）
- **段落间用场景空镜过渡** — 本风格有丰富的场景资产（不同时段/天候变体），段落衔接建议用场景空镜做情绪缓冲，不要硬切
- **转折点用视觉而非台词** — 优先使用画面手段（光影突变、景别跳切、空镜隐喻）而非依赖对白解释

## 五、魔幻乐器与环境音

魔幻世界观下的声音元素约束：

### 乐器选择

- **大提琴** — 悲伤、厚重情感段落的基底乐器，最能表现深沉哀婉
- **竖琴** — 空灵、神秘、仙境段落，晶莹剔透的拨弦适合表现魔法与梦境
- **合唱（人声）** — 情感高潮、神圣时刻、史诗段落，人声烘托磅礴感
- **钢琴** — 开场定调 / 平稳段落 / 内心独白，与弦乐搭配使用
- **小提琴** — 情感激荡、激昂段落，高音表现命运转折
- 弦乐铺底可增加电影感但不宜喧宾夺主

### 乐器组合策略

| 情绪阶段 | 乐器组合 |
|---|---|
| 平稳/开场/收尾 | 钢琴独奏 或 钢琴 + 竖琴 |
| 神秘渐浓 | 竖琴 + 大提琴 |
| 情绪爆发/命运转折 | 合唱 + 小提琴 或 管弦乐全奏 |
| 紧张/急促 | 小提琴快速拨弦 + 弦乐急促衬底 |

### 魔幻环境音

- **典型环境音层次** — 风声穿过树林 / 溪水潺潺 / 篝火噼啪 / 马蹄踏草 / 夜雨滴檐 / 布料摩擦 / 魔法嗡鸣
- **每场戏标注 1-2 个核心环境音**，帮助后续音效设计。环境音层次越丰富，魔幻场景越有沉浸感
