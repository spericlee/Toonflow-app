# 真人魔幻写实风格说明

本风格专为"真人魔幻写实"题材打造，所有美术提示词、规范和生成内容均严格限定于：

- **魔幻世界观**：以西方古典魔幻为背景，涵盖城堡、森林、宫殿、魔法器物、奇幻服饰、神秘生物等，杜绝现代科技元素。
- **真人写实**：所有角色、场景、道具均以真人写实摄影为标准，强调五官、皮肤、发丝、材质等极致细节与真实质感。
- **魔幻写实纪实**：在写实基础上追求史诗级电影氛围感，画面明暗对比强烈，细节丰富，整体氛围神秘中见华美、冷冽中蕴暖光。
- **浪漫奇幻**：适合展现宿命、冒险、禁忌之恋的魔幻故事，强调西方美学下的人性光辉与魔法情感表达。

## 适用范围

- 真人魔幻题材的AI美术生成（角色、场景、道具、分镜等）
- 需要极致写实、细腻质感、史诗级魔幻氛围的视觉创作

## 严禁内容

- 卡通、动漫、插画、二次元风格
- 现代元素、科幻元素、赛博朋克
- 高饱和荧光色、霓虹色
- 暗示性、暴露、透视等不适宜内容

## 风格体验

在本风格下，您将体验到：

- 角色面容精致、气质神秘或高贵，服饰、妆容、发型均为西方魔幻写实
- 场景空间层次丰富，材质纹理超清晰，氛围真实且富有史诗感
- 道具、法器、饰品等均为魔幻造型，工艺精细，质感极致
- 分镜与视频均以真人写实为基础，强调叙事与情感表达

---

本风格仅适用于"真人魔幻写实"相关项目。如需其他风格（如插画、动漫、现代等），请勿使用本套提示词。
